# ✊ Stone Paper Scissors

A simple command-line implementation of the classic **Stone, Paper, Scissors** game built with Python.

The game allows the player to compete against the computer in a **Best of 3** match while keeping track of scores and announcing the overall winner.

---

## ✨ Features

- ✅ Best of 3 gameplay
- ✅ Random computer moves
- ✅ User input validation
- ✅ Automatic score tracking
- ✅ Displays the winner of each round
- ✅ Announces the overall winner
- ✅ Beginner-friendly and well-structured code

## 🚀 Getting Started

### Requirements

- Python 3.6 or later

No external libraries are required.

---

### Installation

Clone the repository:

```bash
git clone https://github.com/MayankBisht-24/Code-Vault-Python.git
```

Go to the project folder:

```bash
cd Code-Vault-Python/Mini-Projects/Stone-Paper-Scissors
```

Run the program:

```bash
python stone_paper_scissors.py
```
